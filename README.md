# Canopy 🌱

![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=for-the-badge&logo=Kotlin&logoColor=white)

Canopy is a communal system for visualizing commuting-related emissions.
The application uses automatic and manual commuting data to display
collective impact through a shared virtual tree.

---

## Project Context
This project is developed as part of a Metropolia's multidiciplinary innovation project course and is heavily under development.

---

## Features
- **Automatic location detection** - With a press of a button, you can satrt automatically detecting your location.
- **Automatic distance detection** - With the help of the location detection, the app can calculate how much you have moved in meters.
- **Transportation type** - Can detect automatically whether you are traveling using a vehicle or walking.
- **Visual frontend** - A nice, greenish UI with visual tree and graphs about your carbon footprint.
